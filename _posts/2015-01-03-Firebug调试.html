日志输出

console.log();
console.warn();
console.error();

定时器调试

console.time("");
console.timeEnd("");

日志分组

console.group("");
console.groupEnd("");
console.dir(); // 输出对象全部属性及方法
console.dirxml(); // 显示当前元素的代码结构

断言

console.assert(); // 断言 console.assert(20==="20"); 失败
console.trace(); // 比如函数执行调用情况

性能

console.profile();
console.profileEnd();
注：其他浏览器使用firebug导入 firebug-lite
